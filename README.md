# edraw-ai-mcp-server

### 什么是 Edraw AI MCP 服务？

**EdrawAIMCP** 是由万兴科技开源的**图表可视化解决方案**，旨在通过**自然语言、图像、文档与代码**四种输入方式，帮助用户**一键生成和智能优化各类可视化图表**。它适用于开发者、产品团队、内容运营人员、教学人员等，能够快速产出高质量的视觉内容，如：

- 流程图  
- 思维导图  
- 时间线  
- 海报  
- PPT 单页  
- 框架图  
- 知识卡片  
- SWOT 分析  
- PEST 分析  
- 精益画布  
- 用户画像  
- 用户故事
